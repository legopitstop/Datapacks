# [<](../README.md) Convert Traders

![alt](banner.png)

Easily convert Wandering Traders to normal Villagers!

## Pack Data

| Key                | Value    |
| ------------------ | -------- |
| supported          | `yes`    |
| namespace          | `trader` |
| pack_version       | `1.2.0`  |
| mc_version         | `1.18.1` |
| multiplayer_tested | `yes`    |
| has_config         | `yes`    |
