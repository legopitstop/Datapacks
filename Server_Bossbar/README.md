# [<](../README.md) Server Bossbar

![alt](pack.png)

This datapack allows you to easily create a custom bossbar for server/map usage!

## Pack Data

| Key                | Value     |
| ------------------ | --------- |
| supported          | `no`      |
| namespace          | `bossbar` |
| pack_version       | `1.4.0 `  |
| mc_version         | `1.16.5`  |
| multiplayer_tested | `no`      |
| has_config         | `yes`     |
